## 문제 링크

[9996 - 한국이 그리울 땐 서버에 접속하지](https://www.acmicpc.net/problem/9996)

## 사용 알고리즘

- 문자열

## 시간 복잡도

- $O(N)$

## 문제 풀이

풀이 과정은 아래와 같습니다.

1. 패턴에서 `*`의 위치를 찾음
2. `*`의 왼쪽 문자열과 오른쪽 문자열을 찾음
3. 입력 문자열을 순회
   1. 입력 문자열의 길이가 패턴의 왼쪽 문자열과 오른쪽 문자열의 길이보다 작으면 `NE`를 결과에 추가
   2. 입력 문자열의 왼쪽 문자열과 오른쪽 문자열이 일치하면 `DA`를 결과에 추가
   3. 그 이외의 경우에 `NE`를 결과에 추가
4. 결과 출력

### C++

```c++ title="C++"
#include <bits/stdc++.h>
using namespace std;

int n, pos;
string p, pre, post, input, ret;

int main() {
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  cout.tie(NULL);

  cin >> n >> p;

  pos = p.find("*");
  pre = p.substr(0, pos);
  post = p.substr(pos + 1);

  for (int i = 0; i < n; i++) {
    cin >> input;

    if (input.size() < pre.size() + post.size()) {
      ret += "NE\n";
      continue;
    }

    if (input.substr(0, pre.size()) == pre &&
        input.substr(input.size() - post.size()) == post) {
      ret += "DA\n";
    } else {
      ret += "NE\n";
    }
  }

  cout << ret;

  return 0;
}
```

### JavaScript

```js title="JavaScript"
const fs = require('fs');
const filePath = process.platform === 'linux' ? '/dev/stdin' : './input.txt';
const input = fs.readFileSync(filePath).toString().trim().split('\n');

input.shift();
const pattern = input.shift();
let ret = '';

const pos = pattern.indexOf('*');
const pre = pattern.substring(0, pos);
const post = pattern.substring(pos + 1);

for (s of input) {
  if (s.length < pre.length + post.length) {
    ret += 'NE\n';
    continue;
  }

  if (
    s.substring(0, pre.length) === pre &&
    s.substring(s.length - post.length) === post
  ) {
    ret += 'DA\n';
  } else {
    ret += 'NE\n';
  }
}

console.log(ret);
```

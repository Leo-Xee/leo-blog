## 문제 링크

[1267 - 핸드폰 요금](https://www.acmicpc.net/problem/1267)

## 사용 알고리즘

- 수학

## 시간 복잡도

- $O(N)$

## 문제 풀이

요금제별 비용은 아래와 같이 증가합나디.

- 영식 요금제는 아래와 같이 증가합니다.
  - 0 - 29초 10원
  - 30 - 59초 20원
  - 60 - 89초 30원
  - ...
- 민식 요금제는 아래와 같이 증가합니다.
  - 0 - 59초 15원
  - 60 - 119초 30원
  - 120 - 179초 45원
  - ...

위 내용을 생각해보면 통화 시간을 각 요금제의 기준 시간으로 나눈 몫에 `+1`한 값에 각 요금제의 요금을 곱하면 됩니다.

### C++

```cpp title="C++"
#include <bits/stdc++.h>
using namespace std;

int n, k, Y, M;

int main() {
  ios_base::sync_with_stdio(false);
  cin.tie(NULL);
  cout.tie(NULL);

  cin >> n;

  while (n--) {
    cin >> k;

    Y += (k / 30 + 1) * 10;
    M += (k / 60 + 1) * 15;
  }

  if (Y > M) {
    cout << "M " << M;
  }

  if (Y < M) {
    cout << "Y " << Y;
  }

  if (Y == M) {
    cout << "Y M " << M;
  }

  return 0;
}
}
```

### JavaScript

```js title="JavaScript"
const fs = require('fs');
const filePath = process.platform === 'linux' ? '/dev/stdin' : './input.txt';
const input = fs.readFileSync(filePath).toString().trim().split('\n');

let Y = 0;
let M = 0;

input.shift();
const costs = input[0].split(' ').map((v) => +v);

costs.forEach((cost) => {
  Y += Math.floor(cost / 30 + 1) * 10;
  M += Math.floor(cost / 60 + 1) * 15;
});

if (Y === M) console.log('Y M', Y);
if (Y > M) console.log('M', M);
if (Y < M) console.log('Y', Y);
```

## 문제 링크

[2178 - 미로 탐색](<(https://www.acmicpc.net/problem/2178)>)

## 사용 알고리즘

- BFS

## 시간 복잡도

- $O(N*M)$

## 문제 풀이

BFS를 활용하면 가중치가 동일한 경우에 한해서 최단 거리를 구하는 문제를 풀 수 있습니다.

BFS는 그래프를 레벨별로 순회하기 때문에 구현을 위해 Queue 자료구조, 2차원 좌표를 관리하기 위해서 Pair 자료구조를 사용했으며 풀이 과정은 다음과 같습니다.

1. 시작 좌표를 Queue에 `push`하고 방문처리
2. Queue가 비었는지 확인하고 비었다면 종료
3. Queue의 첫 번째 값의 좌표를 기준으로 4방향 탐색
   1. 오버플로우이거나 이동이 불가한 좌표면 skip
   2. 이미 방문한 좌표이면 skip
   3. `visited` 배열에 현재까지의 거리값 저장
   4. 이동 가능한 다음 좌표를 Queue에 `push`
4. `visited` 배열의 마지막 값 출력 (`count` 변수 대신 `visited` 배열을 활용)

### C++

```cpp title="C++"
#include <bits/stdc++.h>
using namespace std;

const int max_len = 104;
int dy[] = {-1, 0, 1, 0};
int dx[] = {0, 1, 0, -1};
int n, m, a[max_len][max_len], visited[max_len][max_len], y, x;
string s;

int main() {
  cin >> n >> m;
  for (int i = 0; i < n; i++) {
    cin >> s;
    for (int j = 0; j < m; j++) {
      a[i][j] = s[j] - '0';
    }
  }

  queue<pair<int, int>> q;
  visited[0][0] = 1;
  q.push({0, 0});

  while (q.size()) {
    tie(y, x) = q.front();
    q.pop();

    for (int i = 0; i < 4; i++) {
      int ny = y + dy[i];
      int nx = x + dx[i];
      if (ny < 0 || nx < 0 || ny >= n || nx >= m || a[ny][nx] == 0) continue;
      if (visited[ny][nx]) continue;
      visited[ny][nx] = visited[y][x] + 1;
      q.push({ny, nx});
    }
  }

  cout << visited[n - 1][m - 1];
  return 0;
}
```

### JavaScript

```js title="JavaScript"
const fs = require('fs');
const filePath = process.platform === 'linux' ? '/dev/stdin' : './input.txt';
const input = fs.readFileSync(filePath).toString().split('\n');

const solution = () => {
  const dy = [-1, 0, 1, 0];
  const dx = [0, 1, 0, -1];

  const [n, m] = input
    .shift()
    .split(' ')
    .map((v) => +v);

  const a = input.map((v) => v.split('').map((v) => +v));
  const visited = [...Array(n)].map(() => Array(m).fill(0));
  const q = [];

  visited[0][0] = 1;
  q.push([0, 0]);

  while (q.length) {
    const [y, x] = q.shift();
    for (let i = 0; i < 4; i++) {
      const ny = y + dy[i];
      const nx = x + dx[i];
      if (ny < 0 || nx < 0 || ny >= n || nx >= m || a[ny][nx] === 0) continue;
      if (visited[ny][nx]) continue;
      visited[ny][nx] = visited[y][x] + 1;
      q.push([ny, nx]);
    }
  }
  return visited[n - 1][m - 1];
};

console.log(solution());
```

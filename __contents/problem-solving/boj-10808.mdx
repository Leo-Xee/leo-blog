## 문제 링크

[10808 - 알파벳 개수](https://www.acmicpc.net/problem/10808)

## 사용 알고리즘

- 문자열

## 시간 복잡도

- $O(1)$

## 문제 풀이

알파벳 `A`는 `65`, `a`는 `97`의 아스키 코드 값을 가지고 있습니다. C++의 경우 형변환을, JavaScript의 경우 `charCodeAt`을 활용해서 해당 아스키 코드 값을 구할 수 있습니다.

풀이의 과정의 아래와 같습니다.

1. 문자열 길이만큼 순회
   1. 문자열 내 문자의 아스키 코드 값에서 알파벳 `a`의 아스키 코드 값인 `97` 빼기
   2. 해당 값의 배열 인덱스에 `+1` 증감
2. 배열의 요소들 사이에 공백을 추가해서 문자열로 출력

### C++

```c++ title="C++"
#include <bits/stdc++.h>
using namespace std;

int a[26];
string s;

int main() {
  cin >> s;

  for (int i = 0; i < s.size(); i++) {
    a[s[i] - 97]++;
  }

  for (int i : a) cout << i << " ";

  return 0;
}
```

### JavaScript

```js title="JavaScript"
const fs = require('fs');
const filePath = process.platform === 'linux' ? '/dev/stdin' : './input.txt';
const input = fs.readFileSync(filePath).toString().trim();

const a = Array.from({ length: 26 }).fill(0);

const solution = (input) => {
  input.split('').forEach((v) => a[v.charCodeAt() - 97]++);
  return a.join(' ');
};

console.log(solution(input));
```

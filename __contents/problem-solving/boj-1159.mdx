## 문제 링크

[1159 - 농구 경기](https://www.acmicpc.net/problem/1159)

## 사용 알고리즘

- 구현
- 문자열

## 시간 복잡도

- $O(N)$

## 문제 풀이

알파벳 `A`와 `a`의 아스키 코드는 각각 `65`, `97`라는 점과 문자를 아스키 코드값으로 서로 변환하는 방법을 알면 풀 수 있는 문제입니다.

풀이 과정은 아래와 같습니다.

1. 선수의 성 개수만큼 순회
   1. 선수의 성 첫 문자를 숫자로 변환
   2. 변환한 값에서 `a`의 아스키 코드 값을 뺀 인덱스에 해당하는 알파벳 배열의 요소에 `+1` 증감
2. 알파벳 배열 순회
   1. 배열의 요소 값이 5 이상일 경우, 해당 알파벳을 결과 문자열에 추가
3. 문자열 개수가 0이면 `PREDAJA`, 0이 아니면 결과 문제열 출력

### C++

```c++ title="C++"
#include <bits/stdc++.h>
using namespace std;

int n, a[26];
string s, ret;

int main() {
  cin >> n;

  for (int i = 0; i < n; i++) {
    cin >> s;
    a[s[0] - 97]++;
  }

  for (int i = 0; i < 26; i++) {
    if (a[i] >= 5) {
      ret += char(i + 97);
    }
  }

  if (ret.size() == 0) {
    cout << "PREDAJA";
  } else {
    cout << ret;
  }

  return 0;
}
```

### JavaScript

```js title="JavaScript"
const fs = require('fs');
const filePath = process.platform === 'linux' ? '/dev/stdin' : './input.txt';
const input = fs.readFileSync(filePath).toString().trim().split('\n');

const a = Array.from({ length: 26 }).fill(0);
let ret = '';
input.shift();

input.forEach((v) => {
  if (v.length > 0) {
    a[v[0].charCodeAt() - 'a'.charCodeAt()]++;
  }
});

a.forEach((v, idx) => {
  if (v >= 5) {
    ret += String.fromCharCode(idx + 'a'.charCodeAt());
  }
});

console.log(ret.length === 0 ? 'PREDAJA' : ret);
```
